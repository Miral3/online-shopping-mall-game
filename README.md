# online-shopping-mall-game

## 과제
* Vanilla js, HTML, CSS만 이용하여 프로젝트 완성하기
* JSON 파일을 생성하고 JavaScript를 이용하여 동적으로 데이터 받아오기
* 버튼을 이용하여 데이터 필터링 기능

## 소요 시간
6시간 ~ 8시간
